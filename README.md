# Simple Express + SQLite Auth App

## Setup
- Install Node.js 18+.
- In PowerShell run:

```
npm install
```

## Run (development)
```
npm run dev
```
Open http://localhost:3000

## Run (production)
```
npm start
```

## Features
- Sign up with email + password (bcrypt hashed)
- Login, session via cookie
- Logout redirects to thank-you page
- SQLite file at `database.sqlite`
